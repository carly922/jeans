Hello! Thanks for checking out my project!
This is my concept for a website called "Jeans that Fit", which would be a scraper/aggregator for jeans and pants. It would allow for filtering by color, style, size, fit, cut, etc. This would make it way easier to sort through different brands quickly with the attributes that you want. Ideally, it would also allow users to comment on the fit of a specific item to assist other users in finding the best-fitting item for them. I would also like to implement a way for items to be directly parsed into the database using item links.
